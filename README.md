🤖 CodeAlpha Task 4 – Basic Chatbot Using Python

📌 Project Overview

This project was developed as Task 4 during my CodeAlpha Python Programming Internship.

The Basic Chatbot is a command-line application built using Python that interacts with users by responding to predefined messages. It demonstrates the fundamentals of chatbot development using simple Python concepts.

✨ Features

- 👋 Greets the user
- 💬 Responds to basic questions
- 🔄 Runs continuously until the user exits
- ❌ Ends the conversation when the user types "bye"
- 🐍 Simple, clean, and beginner-friendly Python code

🛠️ Technologies Used

- Python 3
- Visual Studio Code

📂 Project Structure

CodeAlpha_Task4_Basic_Chatbot_Python/
│── chatbot.py
│── README.md

▶️ How to Run

1. Clone this repository.
2. Open the project in Visual Studio Code.
3. Make sure Python 3 is installed.
4. Run the following command:

python chatbot.py

🎯 Learning Outcomes

- Python Fundamentals
- Conditional Statements ("if", "elif", "else")
- Loops ("while")
- User Input Handling
- Basic Chatbot Development
